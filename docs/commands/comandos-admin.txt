### GRUPO
--------------------------------------------------------------------------------

COMANDO: !grupo
CATEGORIA: GRUPO
USO:
  • *!grupo* - Exibe os dados atuais do grupo.
PERMISSÃO: Admin do Grupo ou Dono do Bot

COMANDO: !fotogrupo
CATEGORIA: GRUPO
USO:
  • Envie/responda uma *imagem* com *!fotogrupo* - Altera a foto do grupo.
PERMISSÃO: Admin do Grupo ou Dono do Bot

COMANDO: !aviso
CATEGORIA: GRUPO
USO:
  • Responda alguém com *!aviso* - Adiciona um aviso ao membro respondido.
  • Marque alguém com *!aviso* - Adiciona um aviso ao membro marcado.
DETALHES: OBS: Se o membro chegar a 3 avisos ele é adicionado automaticamente a lista negra do grupo.
PERMISSÃO: Admin do Grupo ou Dono do Bot

COMANDO: !silenciar
CATEGORIA: GRUPO
USO:
  • Responda alguém com *!silenciar* - Alterna o silêncio do membro respondido.
  • Marque alguém com *!silenciar* - Alterna o silêncio do membro marcado.
DETALHES: OBS: Use novamente para desmutar o membro.
PERMISSÃO: Admin do Grupo ou Dono do Bot

COMANDO: !rmaviso
CATEGORIA: GRUPO
USO:
  • Responda alguém com *!rmaviso* - Remove um aviso do membro respondido.
  • Marque alguém com *!rmaviso* - Remove um aviso do membro marcado.
PERMISSÃO: Admin do Grupo ou Dono do Bot

COMANDO: !zeraravisos
CATEGORIA: GRUPO
USO:
  • *!zeraravisos* - Remove todos os avisos dos membros do grupo.
PERMISSÃO: Admin do Grupo ou Dono do Bot

COMANDO: !addfiltros
CATEGORIA: GRUPO
USO:
  • *!addfiltros* batata uva - Adiciona as palavras *batata* e *uva* no filtro de palavras.
DETALHES: OBS: Se algum membro enviar uma mensagem contendo uma palavra do filtro, a mensagem dele será deletada automaticamente.
PERMISSÃO: Admin do Grupo ou Dono do Bot

COMANDO: !rmfiltros
CATEGORIA: GRUPO
USO:
  • *!rmfiltros* batata uva - Remove as palavras *batata* e *uva* do filtro de palavras.
PERMISSÃO: Admin do Grupo ou Dono do Bot

COMANDO: !addlista
CATEGORIA: GRUPO
USO:
  • Responda alguém com *!addlista* - Adiciona o numero de quem foi respondido a lista negra e bane em seguida.
  • Marque alguém com *!addlista* - Adiciona o numero de quem foi marcado a lista negra e bane em seguida.
  • *!addlista* +55219xxxx-xxxx - Adiciona o número digitado a lista negra do grupo e bane em seguida.
DETALHES: .
PERMISSÃO: Admin do Grupo ou Dono do Bot

COMANDO: !rmlista
CATEGORIA: GRUPO
USO:
  • Digite *!rmlista 1* - Remove o usuário selecionado da lista negra.
  • *!rmlista* +55219xxxx-xxxx - Remove o número digitado da lista negra do grupo.
DETALHES: Você também pode remover da lista negra da seguinte forma: OBS: Para ver o ID dos usuários é necessário checar no comando *{$p}listanegra*
PERMISSÃO: Admin do Grupo ou Dono do Bot

COMANDO: !listanegra
CATEGORIA: GRUPO
USO:
  • *!listanegra* - Exibe a lista negra do grupo.
PERMISSÃO: Admin do Grupo ou Dono do Bot

COMANDO: !add

  • *!add* +55219xxxx-xxxx, +55119xxxx-xxxx - Digite os numeros com o código do país (adiciona mais de uma pessoa no grupo).

COMANDO: !bcmd
CATEGORIA: GRUPO
USO:
  • *!bcmd* !s !sgif !play - Bloqueia no grupo os comandos !s, !sgif e !play (você pode escolher os comandos a sua necessidade).
  • *!bcmd* sticker - Bloqueia todos os comandos da categoria STICKER.
  • *!bcmd* utilidade - Bloqueia todos os comandos da categoria UTILIDADE.
  • *!bcmd* download - Bloqueia todos os comandos da categoria DOWNLOAD.
  • *!bcmd* variado - Bloqueia todos os comandos da categoria VARIADO.
DETALHES: OBS: Você não pode bloquear comandos de administrador.
PERMISSÃO: Admin do Grupo ou Dono do Bot

COMANDO: !dcmd
CATEGORIA: GRUPO
USO:
  • *!dcmd* !s !sgif !play - Desbloqueia no grupo os comandos !s, !sgif e !play.
  • *!dcmd* todos - Desbloqueia todos os comandos.
  • *!dcmd* sticker - Desbloqueia todos os comandos da categoria STICKER.
  • *!dcmd* utilidade - Desbloqueia todos os comandos da categoria UTILIDADE.
  • *!dcmd* download - Desbloqueia todos os comandos da categoria DOWNLOAD.
  • *!dcmd* variado - Desbloqueia todos os comandos da categoria VARIADO.
DETALHES: OBS: Verifique os comandos que estão bloqueados com *{$p}status*.
PERMISSÃO: Admin do Grupo ou Dono do Bot
