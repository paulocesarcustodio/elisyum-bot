COMANDOS DO BOT ELISYUM
================================================================================


### INFORMAÇÃO
--------------------------------------------------------------------------------

COMANDO: !dbstats
CATEGORIA: INFORMAÇÃO
USO:
  • *!dbstats* - Exibe estatísticas do banco de dados (contatos e comandos).
PERMISSÃO: Usuário

COMANDO: !logs
CATEGORIA: INFORMAÇÃO
USO:
  • *!logs* [quantidade] - Exibe os últimos comandos executados.
PERMISSÃO: Usuário

COMANDO: !contatos
CATEGORIA: INFORMAÇÃO
USO:
  • *!contatos* - Lista os contatos salvos no cache do banco de dados.
PERMISSÃO: Usuário

COMANDO: !menu
CATEGORIA: INFORMAÇÃO
USO:
  • *!menu* - Exibe o menu de comandos gerais.
PERMISSÃO: Usuário

COMANDO: !reportar
CATEGORIA: INFORMAÇÃO
USO:
  • *!reportar* mensagem - Reporta uma mensagem para a administração do Bot.
PERMISSÃO: Usuário

COMANDO: !meusdados
CATEGORIA: INFORMAÇÃO
USO:
  • *!meusdados* - Exibe seus dados gerais como comandos, mensagens, tipo de usuário, etc.
PERMISSÃO: Usuário

COMANDO: !info
CATEGORIA: INFORMAÇÃO
USO:
  • *!info* - Exibe as informações completas do bot, inclusive as configurações atuais.
PERMISSÃO: Usuário


### DOWNLOAD
--------------------------------------------------------------------------------

COMANDO: !d
CATEGORIA: DOWNLOAD
USO:
  • *!d* link - Download automático de mídias do YouTube, Instagram, Facebook, TikTok e Twitter/X.
DETALHES: Também funciona respondendo mensagens com links ou buscando por título no YouTube.
PERMISSÃO: Usuário

COMANDO: !play
CATEGORIA: DOWNLOAD
USO:
  • *!play* musica - Faz download de uma música do Youtube e envia como audio.
DETALHES: Também funciona respondendo mensagens com links do YouTube.
PERMISSÃO: Usuário

COMANDO: !ask
CATEGORIA: DOWNLOAD
USO:
  • *!ask* qual comando faz download de vídeo? - Pergunta sobre comandos disponíveis usando inteligência artificial.
PERMISSÃO: Usuário


### UTILIDADE
--------------------------------------------------------------------------------

COMANDO: !img
CATEGORIA: UTILIDADE
USO:
  • *!img* tema - Envia 2 imagens relacionadas ao tema pesquisado.
PERMISSÃO: Usuário

COMANDO: !revelar
CATEGORIA: UTILIDADE
USO:
  • Responda uma mensagem de *visualização única* com *!revelar* - Revela a imagem/vídeo de visualização única.
DETALHES: OBS: Este comando funciona apenas com mensagens de *VISUALIZAÇÃO ÚNICA* (view once).
PERMISSÃO: Usuário

COMANDO: !save
CATEGORIA: UTILIDADE
USO:
  • Responda um *áudio* com *!save nome-do-audio* - Salva o áudio globalmente para todos usarem.
DETALHES: OBS: Este comando funciona apenas com *ÁUDIOS*. O áudio ficará disponível para todos!
PERMISSÃO: Usuário

COMANDO: !audio
CATEGORIA: UTILIDADE
USO:
  • *!audio nome-do-audio* - Reproduz um áudio salvo.
  • Responda uma mensagem com *!audio nome-do-audio* - Reproduz o áudio como resposta.
PERMISSÃO: Usuário

COMANDO: !audios
CATEGORIA: UTILIDADE
USO:
  • *!audios* - Lista todos os áudios disponíveis (página 1).
  • *!audios 2* - Lista a página 2 dos áudios.
PERMISSÃO: Usuário

COMANDO: !delete
CATEGORIA: UTILIDADE
USO:
  • *!delete nome-do-audio* - Deleta permanentemente um áudio que você criou.
DETALHES: *Atenção*: Só o criador pode deletar! Esta ação não pode ser desfeita!
PERMISSÃO: Usuário

COMANDO: !rename
CATEGORIA: UTILIDADE
USO:
  • *!rename nome-antigo | nome-novo* - Renomeia um áudio que você criou.
DETALHES: Use *|* para separar o nome antigo do novo. OBS: Só o criador pode renomear!
PERMISSÃO: Usuário


### STICKER
--------------------------------------------------------------------------------

COMANDO: !s
CATEGORIA: STICKER
USO:
  • Envie/responda uma *IMAGEM/VIDEO* com *!s* - Transforma em sticker.
  • Responda uma *MENSAGEM DE TEXTO* com *!s* - Transforma em sticker estilo WhatsApp.
  • Envie/responda uma *IMAGEM* com *!s 1* - Transforma em sticker circular.
  • Envie/responda uma *IMAGEM* com *!s 2* - Transforma em sticker sem perder a proporção.
PERMISSÃO: Usuário

COMANDO: !simg
CATEGORIA: STICKER
USO:
  • Responda um sticker com *!simg* - Transforma o sticker em imagem.
DETALHES: OBS: Este comando funciona apenas com *STICKERS NÃO ANIMADOS*.
PERMISSÃO: Usuário


### VARIADO
--------------------------------------------------------------------------------

COMANDO: !vtnc
CATEGORIA: VARIADO
USO:
  • *!vtnc* @membro - Manda o ASCII desejado pro membro mencionado.
  • Responder com *!vtnc* - Manda o ASCII desejado para o membro respondido.
PERMISSÃO: Usuário
